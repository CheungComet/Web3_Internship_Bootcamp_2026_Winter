**学习内容**

| 1、什么是区块链 |
| 区块链是一种去中心化的复盘本技术，通过密码学方法将数据按时间顺序备份成区块，并以链式结构连接，保证数据不可篡改、可追溯。区块链不是依赖单一中心机构，而是由网络中的多个节点共同维护复盘本的一致性。其核心机制包括认知算法（如区块链首先重视比特币，为解决“信任”问题，避免重复记账和双重支付。随着技术的发展，区块链已划分金融、供应链管理、数字身份、物联网等领域。总体而言，区块链通过技术手段降低了信任成本，提高了系统完整性和安全性，但在性能、一致性和监管方面仍面临挑战。 |
| 2，什么是以太坊 |
| 以太坊是一种开源的区块链平台，在开源区块链技术基础上引入了智能合约和去中心化应用（DApp）的概念。与比特币主要为了价值转移不同，以太坊更加强调强化性，通过以太坊虚拟机（EVM）在全球节点上执行统一的代码逻辑。开发者可以使用Solidity等语言编写智能合约，实现去中心化金融（DeFi）、NFT、DAO以太坊采用工作量证明（PoW），后期升级为权益证明（PoS）机制，显着降低了汇率并提升了网络可持续性。总体来看，以太坊构建了一个去中心化的应用生态，为区块链从“数字货币”走向“通用计算平台”奠定了基础，但在扩展性和交易成本方面仍需持续优化。 |
| 2014年通过众筹完成早期融资，2015年主网正式上线。2016年发生DAO事件，社区通过硬分叉以太坊（ETH）与以太坊经典（ETC），凸显了以太坊形成了以社区为核心的治理模式。2017年起，以太坊生态快速扩展，ICO、DeFi、NFT等应用不断支持，成为去中心化应用最活跃的平台之一。为解决机制和融资问题，以太坊启动ETH2.0计划，并于2022年完成Merge，从工作量证明转向权益证明，大幅降低能源消耗。此后，以太坊通过Layer2与扩展方案持续演进，朝着高效率、可持续的区块链基础设施发展。 |
| V神说：我猛然意识到，“以太”这个词是我迄今为止看到的最佳选择，它指的是一种假设在宇宙中并允许光传播的不可见介质。 |
| 3，什么是web3 |
| Web3是基于区块链技术的新一代互联网形态，核心目标是实现中心化、用户主权和价值去回归个人。与Web2中数据和平台由中心化公司掌控不同，Web3区块链、智能合约和加密技术，使用户真正拥有自己的数据、身份和数字资产。其关键特征包括去中心化应用（DApp）、去信任协作、代币机制以及去中心化自治组织（DAO）。Web3不仅关注信息传递，更强调自由流转与透明治理的价值，广泛评估金融（DeFi）、数字内容、游戏和社交等领域。总体而言，Web3被视为互联网从“平台控制”迈向“用户共治”的重要探索，但在技术成熟度、监管和用户体验方面仍处于发展阶段。 |
